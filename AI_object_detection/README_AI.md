# 🤖 AI Object Detection & Data Collection in ROS 2
แพ็กเกจบทเรียนนี้ประกอบด้วยโปรแกรม 2 โหนดหลักที่ใช้กระบวนการเบื้องหลังด้าน AI โดยร่วมกับหุ่นยนต์ใน Webots

## การเตรียมตัว (สำคัญ)
ในการจะรัน AI ต้องมั่นใจว่าเครื่องคุณติดตั้งไลบรารีเสร็จสิ้นแล้ว:
```bash
pip install ultralytics opencv-python
```

---

## ส่วนที่ 1: ระบบดึงข้อมูลสะสมรูปภาพ (Data Collection)
เมื่อเปิดหน้าต่างจำลองหุ่นยนต์ Webots (`ros2 launch webots_diffdrive turtlebot_multi_launch.launch.py`) ขึ้นมาเรียบร้อยแล้ว ให้สั่งรันโหนดเก็บข้อมูลนี้แยกอีกหน้าต่าง
```bash
ros2 run AI_object_detection collect_data
```
- ระบบจะเปิดหน้าต่างจอรับภาพ (Camera View) โผล่ขึ้นมา 
- บังคับหุ่นเดินไปในซิมูเลเตอร์หามุมต่างๆ แล้ว **คลิกเมาส์ให้ Focus อยู่ในหน้าต่าง Camera View**
- จากนั้นให้ **กดแป้นตัวอักษร 'S' บนคีย์บอร์ด** เพื่อให้ระบบสแนปชัตเตอร์ถ่ายภาพ
- กล่องข้อความจะแจ้งว่าภาพถูกเก็บลงที่โฟลเดอร์รันคำสั่ง (`collect_data/` ภายใน Workspace) อัตโนมัติ

*หลังจากได้รูปมุมประณีตเยอะพอแล้ว ให้นำภาพเหล่านั้นไปอัปโหลดเข้าเว็บไซต์ Roboflow วาด Bounding Box คลุมวัตถุ (Train AI) เพื่อ Export ไฟล์ Weight ນามสกุล `.pt` เพื่อใช้ต่อในขั้นที่ 2*

---

## ส่วนที่ 2: ระบบตรวจหาวัตถุ (Object Detection Model inference)
เมื่อทำการเทรนโมเดลผ่าน Roboflow หรือ YOLO สำเร็จ คุณจะได้ไฟล์โมเดลมา (สมมติชื่อไฟล์ว่า `my_best.pt`) ให้ปิดโหนดอันแรกให้หมด แล้วทดสอบรันหน้าต่าง AI นี้:

1. ให้นำไฟล์ `my_best.pt` วางไว้ในรากของโปรเจกต์ (หรือระบุ Path ตามจริงก็ได้)
2. สั่งรันโหนดพร้อมชี้พารามิเตอร์ส่งทางเข้าโมเดลเป้าหมาย:
```bash
ros2 run AI_object_detection detect_object --ros-args -p weights:=my_best.pt
```

*(หากคุณเพิ่งโหลดไฟล์มา ยังไม่เทรน ถ้ารันแบบปกติตามข้างล่าง ระบบจะไปโหลดน้ำหนัก `yolov8n.pt` ดีฟอลต์ที่มีแต่โมเดลบ้านๆ (แมว, หมา, โต๊ะ ฯลฯ) มาลอง Detect เล่นแทนครับ)*
```bash
ros2 run AI_object_detection detect_object
```

- ระบบจะปรากฏจอภาพซ้อนทับกรอบวาดรูป (Bounding Boxes) พร้อมเปอร์เซ็นต์ความมั่นใจ (Confidence Score) ลงบนภาพสิ่งของที่เห็นใน Webots Real-time เลย!
